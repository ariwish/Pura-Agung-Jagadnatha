# Pura Agung Jagadnatha

**Pura Agung Jagadnatha** adalah proyek website untuk memperkenalkan keindahan dan sejarah dari Pura Agung Jagadnatha yang terletak di Kota Denpasar, Bali.

🔗 **Tautan Langsung: [ariwish.github.io/Pura-Agung-Jagadnatha](https://ariwish.github.io/Pura-Agung-Jagadnatha/)**

![Tampilan Project](https://github.com/user-attachments/assets/89cd5df7-dd2b-4ab6-9262-7c5eb5fd14aa)

## ✨ Fitur Utama
*   **Desain Responsif**: Tampilan yang modern dan optimal di berbagai ukuran layar (Mobile, Tablet, Desktop).
*   **Hero Slider**: Transisi gambar beranda yang halus.
*   **Sejarah Interaktif**: Fitur "Baca Selengkapnya" untuk menyajikan sejarah pura secara ringkas namun mendalam.
*   **Galeri Carousel**: Carousel gambar otomatis dengan fitur *Lightbox* untuk melihat foto dalam ukuran penuh.
*   **Swipe Gallery**: Galeri kartu yang interaktif dan mendukung swipe.
*   **Integrasi Peta**: Lokasi yang terintegrasi langsung dengan Google Maps.
*   **Aksara Bali**: Penggunaan font khusus untuk menampilkan keindahan Aksara Bali.

## 🛠️ Tech Stack
*   **HTML5**: Struktur website.
*   **Vanilla CSS3**: Styling menggunakan variabel CSS, Grid, Flexbox, dan animasi.
*   **Vanilla JavaScript**: Semua interaksi (slider, carousel, lightbox) dibangun tanpa menggunakan *library* pihak ketiga.
*   **Google Fonts**: Tipografi menggunakan font *Outfit*, *Playfair Display*, dan *Noto Sans Balinese*.
*   **Font Awesome**: Set ikon untuk navigasi dan elemen sosial media.

## 🚀 Cara Menjalankan
1.  *Clone* repo atau download ZIP file.
2.  Ekstrak file jika download ZIP file.
3.  Buka folder projek.
4.  Klik dua kali pada file `index.html` untuk membukanya di browser.

## 📂 Struktur projek
*   `index.html`: File utama untuk struktur konten.
*   `style.css`: Berisi seluruh desain, tata letak, dan animasi.
*   `scripts.js`: Berisi logika untuk fitur interaktif website.
*   `assets/`: Folder aset seperti logo dan favicon.
*   `images/`: Folder koleksi foto galeri, hero dan gambar latar.
*   `images-swipe/`: Folder koleksi foto untuk swipe gallery.
